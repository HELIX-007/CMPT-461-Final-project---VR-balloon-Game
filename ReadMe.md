🎈 VR Balloon Shooter
Created by: Anush

Course: BSc Computer Science - Final Project

🎮 Game Overview
This is a non-violent, gaze-based VR experience designed for mobile devices. The goal is to pop balloons in a gallery setting across two increasingly difficult levels.

👓 Hardware Requirements
Headset: Optimized for Li-on VR Mini (or any Google Cardboard-compatible headset).

Device: iOS or Android smartphone with a gyroscope.

🛠 Setup & Installation
Host: The game is hosted via GitHub Pages.

Permissions (Crucial): * On iOS, you must grant "Motion & Orientation" access when prompted.

Ensure your phone's physical Silent Switch is OFF to hear the audio.

Browser Setup: Tap the "AA" icon in the Safari address bar and select "Hide Toolbar" to ensure the VR button is visible and the dual-screen view isn't cut off.

🕹 Controls
Movement: Head-tracking (360-degree look-around).

Interaction: Gaze-based. Aim the yellow ring at a balloon. Hold your gaze steady for 1 second. The ring will shrink, triggering a "pop" once the fuse is complete.

Navigation: The 2D Start Screen requires one initial tap to unlock the audio engine and sensors.

🚀 Features & Project Requirements
Two Levels: * Level 1: Stationary targets in a 3x3 grid.

Level 2: Dynamic, moving targets with randomized speed and direction.

Custom Assets: Includes a custom-modeled balloon.glb and textured environment.

Audio: Features background ambient music and spatial 3D sound effects for popping.

HUD Scoreboard: A Head-Up Display is anchored to the player's POV, specifically positioned for stereoscopic clarity in VR lenses.
